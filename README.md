# Circle CI dashboardName

See the status of your Circle CI builds

## How to use

1. Install locally or visit [http://circledashboard.herokuapp.com](http://circledashboard.herokuapp.com).
2. Enter your Circle API token found [here](https://circleci.com/account/api).
3. Hope your builds pass!

### How to Install

1. Clone repo then:
2. `npm i`
3. `npm start`
